# 모임 이벤트 관리 웹앱 MVP PRD

## 핵심 정보

**목적**: 소규모 정기 그룹 모임에서 주최자가 담당하는 공지, 참여자 관리, 카풀 조율, 비용 정산을 한 곳에서 처리
**사용자**: 수영·헬스·친구 모임 등 소규모 동호회를 운영하는 주최자와 참여 멤버

---

## 사용자 여정

```
1. [대시보드]
   ↓ 내 그룹 없음 → "그룹 만들기" 또는 "초대 코드로 참여"

2-A. [그룹 생성 페이지] → 그룹 생성 완료
   ↓ 자동으로 [그룹 홈] 이동

2-B. [초대 코드 입력 페이지] → 코드 검증
   ↓ 성공 → [그룹 홈] 이동 / 실패 → 에러 표시

3. [그룹 홈]
   ↓ 주최자: "이벤트 생성" 클릭

4. [이벤트 생성 페이지] → 이벤트 생성 완료
   ↓ 자동으로 [이벤트 상세 페이지] 이동

5. [이벤트 상세 페이지]
   ↓ 멤버: RSVP 선택 (참석/불참/미정)
   ↓ 주최자: "공지 작성" 클릭

6. [이벤트 공지 페이지] → 공지 작성/확인

   카풀 필요 시 → [카풀 페이지] → 운전자/동승 희망 등록

   모임 종료 후 → [정산 페이지] → 비용 입력 → 균등 분할 금액 확인
```

## 기능 명세

### 1. MVP 핵심 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|--------------|------------|
| **F001** | 그룹 생성 | 이름·설명·최대 인원 입력 후 그룹 개설, 8자리 초대 코드 자동 발급 | 서비스의 기본 단위 | 그룹 생성 페이지 |
| **F002** | 초대 코드 가입 | 8자리 코드 입력 → 멤버로 그룹 합류 | 멤버 모집 유일 수단 | 초대 코드 입력 페이지 |
| **F003** | 그룹 멤버 목록 조회 | 그룹 내 멤버와 역할(owner/admin/member) 표시 | 구성원 파악 필수 | 그룹 멤버 페이지 |
| **F004** | 이벤트 생성 | 일시·장소·최대 인원 설정 후 이벤트 개설 | 모임의 핵심 단위 | 이벤트 생성 페이지 |
| **F005** | 이벤트 RSVP | 참석/불참/미정 선택 + 메모 입력 | 참석자 현황 파악 | 이벤트 상세 페이지 |
| **F006** | 공지 작성·조회 | 그룹/이벤트 단위 공지 게시, 핀 고정 지원 | 준비물·변경사항 전파 | 그룹 공지 페이지, 이벤트 공지 페이지 |
| **F007** | 카풀 등록 | 운전자(빈 좌석 수·출발지) 또는 동승 희망 등록 | 이동 조율 핵심 기능 | 카풀 페이지 |
| **F008** | 정산 생성·조회 | 총 비용 입력 → 참석자 수 기준 균등 분할 금액 산출 | 모임 후 비용 처리 | 정산 페이지 |
| **F009** | 정산 납부 처리 | 개별 납부 여부 체크 (is_paid 토글) | 정산 완료 추적 | 정산 페이지 |

### 2. MVP 필수 지원 기능

| ID | 기능명 | 설명 | MVP 필수 이유 | 관련 페이지 |
|----|--------|------|--------------|------------|
| **F010** | 기본 인증 | 이메일/비밀번호 회원가입·로그인·로그아웃 (Google OAuth 포함, 기존 구현) | 서비스 이용 최소 인증 | 로그인 페이지, 회원가입 페이지 |
| **F011** | 프로필 관리 | display_name·bio 수정 (기존 구현) | 멤버 식별 필수 | 프로필 페이지 |
| **F012** | 대시보드 | 내가 속한 그룹 목록 및 다가오는 이벤트 요약 표시 | 서비스 진입점 | 대시보드 페이지 |
| **F013** | 그룹 설정 | 그룹명·설명·최대 인원 수정, 초대 코드 재발급 (owner/admin 전용) | 그룹 운영 필수 | 그룹 설정 페이지 |

### 3. MVP 이후 기능 (제외)

- 이메일·푸시 알림 (공지 발행 시 알림)
- 카풀 자동 매칭 알고리즘
- 항목별 차등 정산 (더치페이 앱 수준)
- 사진 첨부 (이벤트·공지)
- 통계·리포트 (출석률, 지출 내역)
- 그룹 공개 검색

└── 로그아웃 - F010

[비로그인]

## 페이지별 상세 기능

### 대시보드 페이지

> **구현 기능:** `F012` | **경로:** `/protected`

| 항목 | 내용 |
|------|------|
| **역할** | 서비스 메인 진입점. 내 그룹 목록과 다가오는 이벤트를 한눈에 파악 |
| **진입 경로** | 로그인 완료 후 자동 리디렉션, 상단 홈 아이콘 클릭 |
| **사용자 행동** | 내 그룹 카드 클릭 → 그룹 홈으로 이동, 그룹 만들기/참여 버튼 클릭 |
| **주요 기능** | - 내가 속한 그룹 카드 목록 (이름, 멤버 수, 다음 이벤트 일시)<br>- 7일 이내 예정 이벤트 요약 리스트<br>- **그룹 만들기** 버튼 (그룹 생성 페이지 이동)<br>- **초대 코드로 참여** 버튼 (초대 코드 입력 페이지 이동) |
| **다음 이동** | 그룹 카드 클릭 → 그룹 홈 페이지, 버튼 클릭 → 각 해당 페이지 |

### 그룹 생성 페이지

> **구현 기능:** `F001` | **경로:** `/protected/groups/new`

| 항목 | 내용 |
|------|------|
| **역할** | 새 그룹을 개설하고 8자리 초대 코드를 자동 발급 |

### 그룹 홈 페이지

> **구현 기능:** `F003`, `F004`, `F006`, `F013` | **경로:** `/protected/groups/[groupId]`

| 항목 | 내용 |
|------|------|
| **역할** | 그룹의 메인 허브. 이벤트·공지·멤버·설정으로 진입하는 중심 페이지 |
| **진입 경로** | 대시보드 또는 내 그룹 목록에서 그룹 카드 클릭 |

| **다음 이동** | 이벤트 카드 → 이벤트 상세 페이지, 이벤트 생성 버튼 → 이벤트 생성 페이지 |
